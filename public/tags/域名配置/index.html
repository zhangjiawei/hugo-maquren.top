购买好域名后如何白嫖 Cloudflare：从注册到完整配置实战

一、前言：为什么一定要用 Cloudflare？ 大多数人买完域名，只停留在“能访问就行”。 但对开发工程师/内容创作者来说，更理想的状态是： 全站 HTTPS 加密，访问更安全、权重更友好； 有一层 全球 CDN 加速和 DDoS 防护； 可以方便地绑定各种服务（Pages、Workers、邮箱、对象存储等）； 最好是 完全免费、不自己运维服务器。 Cloudflare 的免费套餐正好满足这些需求。 这篇文章就是：从你已经买好域名开始，教你如何“白嫖” Cloudflare 的核心能力，并把域名配置完整。 本文示例以 maquren.top 为例，域名注册商以 NameSilo 为例，其他注册商（阿里云、腾讯云等）操作类似。 二、前置条件：你需要准备什么 开始之前，请确认你已经具备： 一个已购买的域名（例如：maquren.top），能登录到注册商后台（如 NameSilo）； 一个 Cloudflare 账号，如果没有可以先免费注册； 能访问 Cloudflare 控制台（https://dash.cloudflare.com）； 对 DNS 解析有基础概念（知道 A 记录 / CNAME 的大致含义即可）。 如果你还没有 Cloudflare 账号： 打开 https://dash.cloudflare.com； 使用邮箱注册一个免费账号（Free Plan 就够用）； 登录后即可开始添加站点。 三、第一步：在 Cloudflare 添加你的域名 登录 Cloudflare 控制台； 点击右上角的 “Add a site” / “添加站点”； 输入你的域名（例如 maquren.top），注意不要带 http:// 或子域名； 选择套餐时，直接选 Free（免费版）； 继续后，Cloudflare 会尝试扫描你当前的 DNS 记录： 如果是新域名，基本上没有什么记录； 如果之前在别的地方已经有解析，可以先保留这些记录，后面再精简。 建议： ...
